# Orwig Tax Services — Website

A simple static website (HTML/CSS/JS only — no build step, no server required).

## Files

- `index.html` — page content (edit text here)
- `css/style.css` — colors, fonts, spacing
- `js/script.js` — small interactive bits (mobile menu, footer year)
- `images/` — put photos/logo files here, then reference them in `index.html`

## Preview it locally

Just double-click `index.html` — it opens in your browser. No install needed.

## Publish updates

This site is hosted on GitHub Pages. To publish a change:

1. Edit the file(s) you want to change and save.
2. Open a terminal in this folder and run:
   ```
   git add .
   git commit -m "describe your change"
   git push
   ```
3. Wait about a minute — GitHub Pages rebuilds automatically at the live URL.
